# エントリーシグナル仮説空間

Created: 2026-03-30
Purpose: 4 Agent並列検証の入力。全カテゴリを網羅的に列挙する。
Method: ADR-013に基づく。

## Zwicky Box（次元分解）

全仮説は以下の次元の組合せで構成される。

### シグナル次元（何をシグナルとするか）

| 次元 | パラメータ値 |
|------|------------|
| A. シグナル源 | 62カテゴリ（Cat 1-67。旧Cat 10/15/21/22はメタ次元に移動、Cat 30は統合カテゴリ） |
| B. シグナルの読み方 | 水準 / 変化量 / モメンタム / 極値(平均回帰) / ブレイクアウト / パターン |
| C. シグナルの時間軸 | 日中 / 日次 / 複数日 / 週次 |
| D. エントリー方向ロジック | 順張り / 逆張り / 条件付き切替 |

### 執行次元（どう実行するか）— Stage 2以降で展開

| 次元 | パラメータ値 |
|------|------------|
| E. エントリータイミング | 開場前指値 / 始値成行 / 60分後 / 引け前 |
| F. 出口方法 | 時間ベース / 価格ターゲット / トレーリング / ボラティリティベース |

### メタ次元（判断・サイジングの枠組み）— シグナルとは別に評価

| カテゴリ | 内容 |
|---------|------|
| M1: アンサンブル/多数決（旧Cat 10, 15） | 複数シグナルの同時発火カウント、多数決、加重平均 |
| M2: 確率×ペイオフの期待値判断（旧Cat 21） | 二値ではなく確率出力。確率×期待ペイオフで判断。着想: 天気予報の降水確率 |
| M3: シグナル強度連動ポジションサイズ（旧Cat 22） | Kelly基準。確信度に比例した段階的サイジング。着想: ポーカーのベットサイズ |

全組合せが有意味ではない。無意味な組合せは各Agentが除外し、除外理由を記録する。
有効な読み方（次元B）の数はシグナル源により異なる。Agent実行時に無意味な組合せを除外し実数を記録する。

## シグナルカテゴリ一覧（25カテゴリ）

### 調査ベース（Cat 1-9）: ドメイン知識 + Web調査12件から

**Cat 1: 価格パターン（~10サブシグナル）**
- ギャップ（寄付 vs 前日終値）のサイズと方向
- 連続上昇/下落日数（2,3,4,5日）
- N日高値/安値からの距離（5,10,20,50日）
- 移動平均乖離率（5,10,20,50日MA）
- RSI（7,14日）
- ボリンジャーバンド位置（上限/下限/中央からの距離）
- ATR比率（当日ATR vs N日平均ATR）
- 日中レンジ vs ATR
- Opening Range（開場60分の高安）
- 終値位置（日中レンジ内のどこで引けたか）

**Cat 2: 出来高（~5サブシグナル）**
- 出来高スパイク（vs 5,10,20日平均）
- 出来高トレンド（N日間の増減傾向）
- 価格-出来高ダイバージェンス（価格上昇+出来高減少等）
- 時間帯別相対出来高（開場/引け/ランチ）
- VWAP乖離（5分足データから算出）

**Cat 3: ボラティリティ（~7サブシグナル）**
- VIX水準
- VIX日次変化量
- VIX期間構造（VIX/VIX3M比率）
- VIX期間構造の変化（前日比）
- 実現Vol vs インプライドVol差（日足リターンから算出した実現Vol vs VIX）
- ATR変化率（ATRの加速/減速）
- 日中ボラティリティパターン（5分足から計算）

**Cat 4: クロスアセット（~5サブシグナル）**
- Brent原油水準/日次変化
- HYスプレッド水準/日次変化
- イールドカーブ水準/日次変化
- USD Index水準/日次変化
- US10Y水準/日次変化
- 学術根拠: 原油ボラティリティは株式リターンの強い予測因子(Journal of Banking & Finance)
- 学術根拠: クレジットスプレッドは株式ストレスの先行指標(RIA)

**Cat 5: カレンダー/行動ファイナンス（~5サブシグナル）**
- 曜日効果（特に月曜午後の反転パターン）
- 月末/月初効果（最終2日+最初2日）
- 祝前日効果
- 月次季節性
- オーバーナイトギャップ方向と大きさ
- 注意: 2001年研究「カレンダー効果はデータドレッジング」。VIXレジーム別で15pp差異(SPY ORB研究)

**Cat 6: レバETF固有（~4サブシグナル）**
- Bull/Bearペアスプレッド（SOXL/SOXS比率の変動）
- ボラティリティ・ディケイ率（理論値 vs 実際の乖離）
- 引け前リバランスフロー方向と規模（レバETFは毎日引けで強制リバランス→制度的に予測可能なフロー。Cat 14と統合して検証）
- インバースETFのプレミアム/ディスカウント
- ※トラッキングエラー（対SOX指数）はベンチマークデータ未保有のため現時点で検証不可
- 学術根拠: Swiss Finance Institute「レバETFリバランスフローが引け前モメンタムと翌日平均回帰を誘発」

**Cat 7: レジーム切替メタ戦略（~5サブシグナル）**
- 高VIX→平均回帰、低VIX→モメンタム（学術的裏付けが複数あり: Economic Modelling, J. of Asset Management）
- レジーム遷移検出（高Vol入り/出の変化点）
- バックワーデーション/コンタンゴ as 方向シグナル
- クレジットレジーム×テクニカルシグナルの交互作用
- 全指標統合レジームスコア × 個別シグナル
- 学術根拠: 「モメンタム利益は先行市場ボラティリティが高いとき崩壊。切替戦略がクラッシュ損失を排除」

**Cat 8: センチメント（~3サブシグナル）**
- VIX期間構造 as センチメント指標（バックワーデーション=短期恐怖が中期を上回る）
- VIXY出来高 as リテールセンチメント代理
- VIXY価格変動 vs VIX実際の変動の乖離
- ※プットコール比率は手元データになし。VIXからの推定は不正確なため除外

**Cat 9: 古典手法の現代適用（~3サブシグナル）**
- Wyckoff蓄積/配分パターン（出来高-価格分析。大口の建玉蓄積→放出を検出）
- スプリング/アップスラスト（サポート/レジスタンスの偽ブレイクアウト後の反転）
- Effort vs Result（出来高が大きいのに価格変化が小さい=吸収。逆は枯渇）
- 根拠: 「Wyckoff法は75年経っても有効。出来高に焦点を当てる点がMA crossover/RSIダイバージェンスより先に隠れた関心を明かす」

### SCAMPER適用（Cat 11-13）

**Cat 11: 相対シグナル** [SCAMPER: Substitute — 絶対値の代わりに相対値を使う]
- SOXL/TQQQ比率の変動（セクター間相対強度）
- Bull/Bear乖離率（SOXL日次リターン vs -1*SOXS日次リターンの差）
- レバETF間のモメンタム格差（直近N日リターンの銘柄間順位変動）

**Cat 12: クロスタイムフレーム** [SCAMPER: Combine — 異なる時間軸のシグナルを組み合わせる]
- 週足シグナル→日足エントリー（週足で上昇トレンド中に日足で押し目買い等）
- 日足シグナル→日中エントリー（日足でシグナル発生後、日中の最適タイミングで執行）
- 複数時間軸の一致/不一致（日足と5分足のモメンタムが同方向/逆方向）

**Cat 13: レジーム遷移** [SCAMPER: Adapt — 他市場のレジーム変化を適用]
- risk_off→risk_on変化点の検出（レジームスコアの変化率）
- VIXレジーム切替の速度（急変 vs 緩慢で後続パターンが異なるか）
- レジーム滞在日数（長期risk_offからの脱出 vs 短期risk_offで異なるか）

### Zwicky Box次元展開で発見（Cat 14, 16-17）

**Cat 14: 予測可能フロー/構造イベント**
- レバETF引けリバランスフロー方向と規模（Cat 6と統合検証）
- オプション満期集中日（OpEx: 毎月第3金曜）前後のパターン
- 四半期末リバランス（機関投資家の定期リバランスフロー）
- 着想: 潮汐は天体力学で予測可能なフロー。同様にOpEx/四半期末/ETFリバランスは制度的に日程が決まっており予測可能（ビソシエーション手法）

**Cat 16: 逆シグナル**
- 「一貫して負ける」シグナルを発見し、逆方向に利用する
- 損失シグナルの反転は利益シグナルと同等の価値を持つ
- Stage 1で方向一致率が有意に50%未満のシグナルを抽出し、反転して再評価

**Cat 17: 伝播遅延（リード・ラグ）**
- 原油変動→VIX→レバETFの時間差（日次ラグ1-3日を検証）
- マクロ指標変化→価格反映の遅延
- クロスアセットのGranger因果性検定で有意なリード・ラグ関係を特定

### 逆ブレインストーミング由来（Cat 18）

**Cat 18: ノートレード条件（悪い日を避けるフィルタ）**
- 「確実に負ける方法」を列挙し、反転して得た着想
- 「全日ロング」から最悪の20%の日を避けるだけでパフォーマンスは改善するか
- 特定条件下でトレードを禁止するフィルタの価値を定量化する
- 候補条件: イベント前日、VIX極端値（>35）、出来高異常低、連続損失後

### 前提反転由来（Cat 19-20, 26-28）

**Cat 19: 方向切替戦略（常時ポジション保有）**
- 前提「エントリーとエグジットを判断する」を反転
- 常にロングまたはショートを保有し、切替タイミングのみ判断する
- 「いつ入るか」ではなく「今どちら向きか」にフレーミングを変える

**Cat 20: 方向予測不要戦略**
- 前提「方向を予測しないと利益が出ない」を反転
- Bull/Bear同時保有ペアトレード（追跡誤差の非対称性を収益化）
- ボラティリティ・ディケイ自体を収益化する戦略
- ※VIXY空売り等の具体的執行手段はサクソバンクでの実行可否要確認

**Cat 26: 1ルール戦略のベースライン**
- 前提「複雑なシステムが単純なルールに勝つ」を反転
- 「VIX>30なら買わない」だけのパフォーマンス
- 「前日が-3%超なら翌日ロング」だけのパフォーマンス
- 最もシンプルなルールが複雑なシステムに勝つか検証する

**Cat 27: 極端条件限定戦略（高選択性）**
- 前提「毎日トレード機会を探すべき」を反転
- 年20回程度しか発動しない厳格な条件でのみトレード
- 候補: σ偏差-3以上 + VIXスパイク + 出来高2倍超の同時発生
- 低頻度だが高確信度の機会のみを狙う

**Cat 28: 純粋マクロ戦略（価格データ不使用）**
- 前提「テクニカル分析が必要」を反転
- レジームスコア + イベント密度だけで方向判断する
- テクニカル分析を完全に排除した場合のパフォーマンスを検証

### ランダム刺激・ビソシエーション由来（Cat 23-25, 29-30）

**Cat 23: Bull/Bear追跡誤差の非対称性**
- Bull ETF（SOXL）とBear ETF（SOXS）は理論上鏡像関係だが、追跡誤差は対称とは限らない
- 日次リターンの非対称性: 上昇日と下落日でディケイ率は異なるか
- 非対称性にエッジがあればペアトレード（Cat 20）の根拠になる
- 着想: ランダム刺激手法。「鏡」から「Bull/Bearの鏡像関係」を連想

**Cat 24: リバランスフィードバックループ検出**
- レバETFのリバランスが市場を動かし、それがさらにリバランスを誘発する自己強化パターンの仮説
- 特定の価格変動幅（例: 日次±5%超）を超えるとフィードバックが増幅するか
- 増幅パターンの検出は翌日の平均回帰の強さを予測する可能性
- 着想: ランダム刺激手法。「合わせ鏡の無限反射」からフィードバックループを連想

**Cat 25: 日中マイクロパターンの崩壊検出**
- 通常の日中パターン: U字型出来高（開場・引けで高、中盤で低）
- このパターンが崩れる日（中盤に異常な出来高等）の特徴を検出
- パターン崩壊が当日後半や翌日の方向を予測するか
- 着想: de Bono Po手法。「市場は毎日同じことを繰り返す」という挑発的仮定の反証から着想

**Cat 29: 減衰振動モデル**
- 物理の減衰振動（バネ）を金融に適用する仮説
- 急落後のリバウンドは振幅が減少し周期が一定のパターンに従うか
- 振幅と周期が過去のパターンから予測可能か。2回目以降のバウンスの方向と大きさ
- 着想: ビソシエーション手法。物理の減衰振動モデルと急落後の価格反発パターンを交差

**Cat 30: 予測可能フローカレンダー（Cat 5, Cat 14と部分重複あり — 統合検証）**
- Cat 5のカレンダー効果 + Cat 14の構造イベントを統合的に検証するカテゴリ
- OpEx（毎月第3金曜）前後のパターン
- 四半期末リバランス
- ETF設定/解約フローの季節性
- Cat 5/14と重複するサブシグナルは一元管理し、重複検証を避ける

### TRIZ・シネクティクス・コンセプトファン・スターバースティング・アトリビュートリスティング由来（Cat 31-39）

**Cat 31: スケールイン/ポジション管理**
- TRIZ原則#1（分割）: ポジションを分割して段階的に構築する
- 1/3をシグナル発生時、1/3を確認後、1/3を押し目（より良い価格）で
- 「一度に全量エントリー」の前提を崩す。部分エントリーのパフォーマンス検証
- 着想: TRIZ（分割原則）+ アトリビュート・リスティング（エントリー回数属性の変形）

**Cat 32: 動的閾値（ボラティリティ連動パラメータ）**
- TRIZ原則#15（ダイナミズム）: 固定パラメータを環境に応じて変動させる
- 例: RSI oversold閾値 = 30（VIX<20時）/ 20（VIX>25時）
- 移動平均期間、ボリンジャーバンド幅、出来高スパイク閾値等を動的化
- Cat 1-3の主要シグナルに対して動的閾値バージョンを検証
- 着想: TRIZ（ダイナミズム原則）

**Cat 33: 強制フロー/カウンターパーティ分析**
- 「取引の相手は誰か？」という問いから着想
- マージンコール時の強制売り: VIX急騰+出来高急増の組合せで検出可能か
- ETFリバランスの強制フロー: 日次変動幅が大きい日ほどリバランス規模が大きい
- OpEx（オプション満期）時の行使/放棄による強制フロー
- 着想: スターバースティング（WHO — 相手方は誰か）

**Cat 34: レバETF日中NAV乖離パターン**
- レバETFの市場価格とNAV（理論値=3×ベンチマーク日次変動）の乖離
- 極端な値動き日（日次±5%超）に乖離が拡大するか
- 日中のどの時間帯で乖離が最大になるか（開場直後/引け前/ランチ）
- 乖離の方向（プレミアム/ディスカウント）が反転を予測するか
- 着想: スターバースティング（WHEN, WHAT — レバETF固有の現象は何か）
- ※ベンチマーク（SOX指数等）データ未保有。Bull/Bearペア比率で間接推定を検討

**Cat 35: 持続ストレス期間（VIX高止まりN日目）**
- 単にVIXが高い（水準）ではなく、VIXが高い状態が何日続いているか（期間）がシグナル
- 恐怖は自然に減衰する: VIX>25が5日続いた後 vs 1日だけの場合で、翌日の挙動は異なるか
- ストレス持続日数とその後の平均回帰確率の関係
- 着想: シネクティクス（ファンタジーアナロジー — 「明日が見えたら何を知りたいか」）

**Cat 36: シグナル減衰（アルファ・ディケイ）検出**
- 過去に有効だったシグナルが時間とともに効果を失うパターン
- Walk-forward検証の各期間でシグナル効果の時系列変化を追跡
- 減衰しているシグナルを排除し、安定しているシグナルを選好する
- 着想: スターバースティング（WHY — なぜシグナルが効かなくなるか）

**Cat 37: 恐怖の時間減衰（VIXスパイク経過時間）**
- VIXは自然に平均回帰する性質がある（学術的に確立）
- VIXスパイク（前日比+20%超等）からの経過日数をシグナルとして使う
- スパイクから3日目、5日目、10日目で平均回帰の強さは異なるか
- 着想: スターバースティング（HOW — 時間の経過自体が機会を生むメカニズム）

**Cat 38: ヘッジ比率調整（Bull/Bear動的ミックス）**
- Bull（SOXL）とBear（SOXS）を常に両方保有し、比率をシグナルで動的に変える
- 例: risk_off時はBull:Bear = 30:70、risk_on時は70:30
- 完全な方向判断ではなく、確信度に応じた「傾き」で管理する
- 着想: コンセプトファン（問題の再定義 — 「エントリー判断」→「ポジション比率管理」へ拡張）

**Cat 39: 執行の微細最適化**
- 最小保有期間フィルタ: エントリー後2時間以内の決済を禁止（ウィップソー防止）
- シグナル後押し目待ち: シグナル発生をフラグとし、次の日中ディップで実際にエントリー
- 時間ベース損切り: 価格ストップに加え、N時間後に含み損なら決済する時間条件
- 動的銘柄選択: その日最もシグナルが強い銘柄を選んでエントリー（ローテーション）
- 着想: アトリビュート・リスティング（「トレード」の属性を分解し各属性を変形）

### Lotus Blossom・追加ランダム刺激・異分野ビソシエーション由来（Cat 40-47）

**Cat 40: ストップハンティング水準の逆用**
- 他の参加者の損切り注文が集中する価格帯を特定する
- 急落時にその水準を下回った後の反発パターン（ストップ刈り後のリバウンド）
- 日足の直近安値、ラウンドナンバー（$40, $50等）の下をヒゲで刺した後の挙動
- 着想: Lotus Blossom（心理→損失回避。他者の損失回避行動を利用する）

**Cat 41: ドローダウン非線形性**
- 3xレバETFのドローダウンは非線形。$100→$70(-30%)の回復に+42.9%必要
- ドローダウンが一定閾値を超えると「回復不能領域」に入り、平均回帰の前提が崩れる
- ドローダウン深度と反発確率の関係。20%と40%で質的に異なるか
- 着想: Lotus Blossom（制約→3xレバの非線形特性）

**Cat 42: 出来高渋滞とブレイクアウト加速度**
- 特定価格帯に多数のバーが集中する「出来高渋滞」の検出（5分足で同価格帯に10本以上）
- 渋滞帯を抜けた後の加速度（ブレイクアウトの最初の5分の変動幅）が持続性を予測するか
- 着想: ランダム刺激（「渋滞」→解消後に一気に流れるアナロジー）

**Cat 43: 複合異常スコア（市場の発熱度）**
- 各指標（VIX, HYスプレッド, Brent, イールドカーブ等）の「正常範囲」を定義
- 正常範囲外の指標数をカウントした複合スコア（0-6）
- 「3つ以上が同時に異常」な日数が持続している期間 as シグナル
- 着想: ランダム刺激（「体温計」→正常体温と発熱の概念）

**Cat 44: 偽の回復検出（指標間の回復タイミングずれ）**
- VIXが急落しても、HYスプレッドがまだ高い場合は「偽の安心」
- 複数指標が全て正常範囲に戻るまでの日数差が、反落の予測因子か
- 先行して回復する指標と遅行する指標のパターン
- 着想: ランダム刺激（「体温計」→解熱剤で下がっても根本原因が未治療なら再発熱）

**Cat 45: 遅延エントリー（シグナル後N日待ち）**
- シグナル発生日に即座にエントリーするのではなく、1日/2日/3日遅らせた場合のパフォーマンス
- 「種を撒いてすぐ収穫しない」：シグナルはフラグ、最適なエントリーは数日後かもしれない
- 全Stage 1通過シグナルに対して遅延バージョンを系統的に検証
- 着想: ランダム刺激（「種まき」→撒いてすぐ収穫できない、時間が必要）

**Cat 46: Bull/Bear出来高比率の振動周期**
- SOXL出来高/SOXS出来高の比率が周期的に振動するか
- Lotka-Volterra（捕食者-被食者）モデルのように、Bull優位→Bear優位が遅延を伴って循環するか
- 振動の周期が安定していれば、次のピーク/ボトムの時期を推定できる
- 着想: ビソシエーション（生態学のLotka-Volterraモデル × Bull/Bear市場動態）

**Cat 47: 定性（イベント）× 定量（テクニカル）統合**
- DuckDBのイベントデータ（カテゴリ、impact、密度）とテクニカルシグナルの組合せ
- 「イベント密度が高い + テクニカルがoversold」は、どちらか単独より予測力が高いか
- Moneyball的発想: 皆が見ていない指標（イベント密度）のほうが情報優位がある可能性
- 着想: ビソシエーション（Moneyball — 過小評価された指標の発見 × 定性+定量統合）

### 世界文学からのビソシエーション（Cat 48-53）

**Cat 48: リベンジトレード防止（連敗時の自動制御）**
- 連続損失後にポジションサイズが増える「損失追いかけ」パターンの防止
- 直近N回の損益に基づき、連敗時にポジションサイズを自動縮小 or ノートレード
- 連敗回数とその後のトレード勝率の関係を検証（連敗後は判断力が低下するか）
- 着想: ドストエフスキー『賭博者』（1866）。主人公アレクセイの損失追いかけの心理描写

**Cat 49: 結果→シグナル逆引き分析**
- 通常の「シグナル→結果」分析ではなく、「大きな結果（日次±5%超等）→前日にどんなシグナルが出ていたか」を逆方向から分析
- 大きく動いた日の前日の共通パターンを抽出する
- Stage 1で見つからなかったシグナルが逆引きで発見される可能性
- 着想: ルイス・キャロル『鏡の国のアリス』（1871）。白の女王は時間が逆行し、結果を先に経験してから原因が起きる

**Cat 50: 市場サイクル位置推定**
- VIXスパイク→恐慌→回復→楽観→次のスパイクのサイクルを過去データから特定
- 過去のサイクルの長さ・振幅のパターンから、現在がサイクルのどの位置にいるかを推定
- サイクル位置に応じて最適な戦略（平均回帰/モメンタム/ノートレード）を切替える
- Cat 7（レジーム切替）の時間軸を拡張した版。レジームの「相」ではなく「位相」を見る
- 着想: ガルシア=マルケス『百年の孤独』（1967）。ブエンディア家が世代を超えて同じ過ちを繰り返す円環的歴史

**Cat 51: シグナル定期入替メカニズム**
- 有効なシグナルは市場参加者に学習されて効果が消える（アルファ・ディケイ）
- Cat 36（シグナル減衰検出）の発見に基づき、減衰が検出されたシグナルを自動的に除外し、新しいシグナルに入替える仕組み
- 「同じ場所にとどまるためには走り続けなければならない」= シグナルの入替えを止めた瞬間にエッジが消える
- 着想: ルイス・キャロル『鏡の国のアリス』の赤の女王効果。および川端康成『碁の名人』の世代交代（古い手法が新しい市場に敗れる）

**Cat 52: イベント密度の集計統計シグナル**
- 個別イベントへの反応ではなく、「過去N日のイベント密度」「positive/negative比率」等の集計値をシグナルにする
- DuckDBのeventsテーブルから算出。イベント密度が閾値を超えた日の翌日リターン
- 「個人は予測不能だが、集団の統計は予測可能」の原則をイベントデータに適用
- Cat 47（定性×定量統合）のイベント側を独立して検証するカテゴリ
- 着想: アシモフ『ファウンデーション』（1951）。ハリ・セルダンの心理歴史学 — 個人ではなく集団の統計的予測

**Cat 53: 市場リズム変調検出**
- 市場には固有のリズム（ボラティリティの周期、出来高の日中パターン等）がある
- リズムが安定している期間 vs 乱れた期間を検出する
- リズムの「変調」（通常と異なる周期・振幅への変化）がレジーム転換の先行シグナルか
- Cat 29（減衰振動）が振動パターン自体を見るのに対し、Cat 53はパターンの「変化」を見る
- 着想: 宮本武蔵『五輪書』（1645）。「拍子」の概念 — 全てのものに拍子があり、敵の拍子を知り、予期しない拍子で打つ

### 古典文学・漫画・アニメ・映画・哲学・音楽からのビソシエーション（Cat 54-59）

**Cat 54: 制約の厳しさ × 精度の関係**
- 厳しい制約をかけた戦略（「SOXLのみ」「VIX>30の日のみ」「デイトレのみ」）は万能戦略より精度が高いか
- 制約の数（1条件/2条件/3条件）と勝率/Sharpeの関係を系統的に検証
- 少数の厳格な条件の組合せが、多数の緩い条件より優れるか
- 着想: 冨樫義博『HUNTER×HUNTER』。念能力は「制約と誓約」で強くなる。厳しい制約ほど威力が増す

**Cat 55: シグナル失敗条件の特定（「免罪体質」検出）**
- 有効なシグナルが効かない市場状態を積極的に特定する
- Stage 1を通過したシグナルに対して「このシグナルが失敗した日」の共通特徴を抽出
- 失敗条件をフィルタとして追加することで精度を改善できるか
- 着想: アニメ『PSYCHO-PASS』。シビュラシステムには検出できない「免罪体質者」が存在する。どんな予測システムにも盲点がある

**Cat 56: 逆張りのキャリーコスト（待ち時間耐性）**
- 逆張りシグナルが「正しくなるまでの待ち時間」と、その間の含み損の大きさ
- 口座サイズ（5万円）で逆張り戦略の含み損に耐えられるか
- 逆張りシグナルの「遅延（Cat 45）」バージョンでキャリーコストを削減できるか
- 着想: 映画『ビッグ・ショート』。マイケル・バーリは正しい逆張りポジションで2年間損失に耐えた

**Cat 57: 最大損失最小化評価（Sortino比的視点）**
- Sharpe比（上下両方のボラティリティ）ではなくSortino比（下方リスクのみ）でシグナルを評価
- 勝率が同じでも最大損失が小さいシグナルを選好する
- 最大ドローダウンが口座サイズの何%になるかを各シグナルで算出
- 着想: アニメ『銀河英雄伝説』。ヤン・ウェンリー「勝つための最良の方法は、負ける可能性の最も少ない戦い方」

**Cat 58: 小さなエッジの反復複利効果**
- 1回あたりの期待値が小さくても（+0.5%等）、高頻度で反復すれば複利で大きくなるか
- 低エッジ×高頻度 vs 高エッジ×低頻度のパフォーマンス比較
- 手数料・スプレッドとの損益分岐点
- 着想: カミュ『シーシュポスの神話』。毎日の繰り返しに意味を見出す。小さな積み重ねの複利効果

**Cat 59: シグナルが効かない市場状態の分離**
- Cat 55の拡張。全シグナルに共通して効かない「市場の異常状態」を特定する
- 複数シグナルが同時に失敗する日の特徴（イベント種別、Vol水準、流動性等）
- 異常状態を検出したらSkill全体が「ノートレード」を出す上位フィルタ
- 着想: 映画『マトリックス』（バックテストはシミュレーション内の結果）+ 『PSYCHO-PASS』

### 歴史・追加領域からのビソシエーション（Cat 60-62）

**Cat 60: 摩擦後期待値（スリッページ差引き評価）**
- バックテストの期待値から、実際のスリッページ・手数料・為替コストを差し引いた「摩擦後期待値」で評価
- サクソバンクの手数料（最低$1.10×往復）+ 為替コスト（±0.25%）を全シグナルの期待値から控除
- 摩擦後にプラスが維持されるシグナルのみを実用候補とする
- 着想: クラウゼヴィッツ『戦争論』（1832）。「戦場の摩擦」— どんな精密な計画も実行段階で摩擦により崩壊する

**Cat 61: 模倣急増の検出（バブル動態シグナル）**
- 出来高の急増 + 価格上昇の加速 + ボラティリティ上昇の同時発生を「模倣急増」として検出
- EPJ Data Science研究: 暴落前1年で投資家間の模倣行動が劇的に増加する
- レバETFのバブル的動態（3xレバによる増幅効果）は通常ETFより早く検出可能か
- 着想: マッケイ『狂気と幻想の群衆心理』（1841）のバブル3事例 + 現代統計研究。ただしマッケイの記述は誇張との批判あり（Goldgar 2007）、データでの検証が必須
- 関連: Cat 43（市場発熱度）の歴史的根拠。手元データ（VIX、HYスプレッド、出来高）で部分構成可能

**Cat 62: カウンター戦略（急落直後の即座逆張り）**
- 大幅下落（日次-5%超等）の翌日にロングエントリーする純粋なカウンター
- Cat 1（ギャップ）やCat 18（ノートレード）とは異なり、「相手の動きの瞬間に打つ」タイミングの精度が焦点
- 急落の種類（VIX起因/イベント起因/テクニカル起因）により反発パターンが異なるか
- 着想: 格闘技のカウンター — 相手が攻撃した瞬間が最も無防備。急落の売り手が売り切った直後が反発の起点

### 歴史の追加深掘り + 既存着想源の追加深化（Cat 63-67）

**Cat 63: 情報鮮度ベースの判断モード切替**
- 即時取得可能な情報（VIX/Brent via yfinance）と遅延情報（HYスプレッド 1-2日遅延、FRED）を区別
- 「即時情報のみモード」と「全情報モード」で判断を切替え、どちらが精度が高いか比較
- JST制約で米国トレーダーより遅いが、マクロデータの公開は全員同時。遅延の影響を定量化
- 着想: ロスチャイルドのワーテルロー（1815）。伝書鳩による24時間の情報優位が巨額利益を生んだ。情報速度がエッジの源泉

**Cat 64: 空売り圧力の代理検出（インバースETF出来高）**
- SOXS/SQQQ等のインバースETF出来高の急増をベア圧力の代理指標として使用
- SOXS出来高/SOXL出来高の比率（Bear/Bull ratio）の急変がシグナルか
- Bear/Bull ratioの極端値は「売り圧力のピーク」=反転ポイントか
- 着想: VOC市場のイサーク・ル・メール（1608、世界初の空売り）+ デ・ラ・ベガ『混乱の混乱』（1688）の「ベア・シンジケート12段階戦略」

**Cat 65: 再帰的フィードバック加速度**
- 価格下落→VIX上昇→レバETFリバランス売り→さらに価格下落の自己強化ループ
- ループの**加速度**を測定: 2日目の下落幅 > 1日目の下落幅なら加速中
- 加速が減速に転じた瞬間（3日目の下落幅 < 2日目）が反転の先行シグナルか
- Cat 24（フィードバックループ）の拡張版。検出対象が「ループの存在」ではなく「ループの速度変化」
- 着想: ジョージ・ソロスの再帰性理論（1987）。ポンド危機（1992）での自己強化ループの実証

**Cat 66: エグジット遅延シグナル（利確を急がない条件）**
- エントリーだけでなく「いつ出ないか」（保有継続条件）の検証
- トレンドが継続している指標（MA方向、出来高増加、VIX安定/低下）が揃っている間は利確しない
- IFD-OCOのTP水準を動的に引き上げる（トレーリング）条件の特定
- 着想: ジェシー・リバモア「私は座っていることで金を稼いだ。トレードではない」（1923）

**Cat 67: テールリスク明示化（最悪5%の損失算出）**
- 全シグナルに対して、平均パフォーマンスだけでなく最悪5パーセンタイルの損失を算出
- 「平均は良いが最悪ケースで口座が吹き飛ぶ」シグナルを排除する
- CVaR（Conditional Value at Risk）的な評価。Cat 57（Sortino/最大損失最小化）の定量版
- 着想: LTCM崩壊（1998）。ノーベル賞受賞者のモデルが極端な市場で破綻。レバレッジ25倍の教訓

## サイズ見積もり

### 計算方法

1. 25シグナルカテゴリの各サブシグナルを列挙 → シグナル源の総数
2. 各シグナル源に対して有意味な読み方（次元B）をAgentが判断し展開
3. 有意味な組合せに対して方向（順張り/逆張り）を展開

### 暫定見積もり（Agent実行前。実数はAgent実行後に確定）

| 段階 | 計算 | 推定数 |
|------|------|--------|
| シグナル源 | 62カテゴリ（Cat 1-67。旧Cat 10/15/21/22はメタ、Cat 30は統合） × 平均3.3サブシグナル | ~205 |
| × 読み方（次元B） | 有意味な読み方はシグナルにより1-6。Agentが判断 | ~500-700 |
| × 方向（次元D） | 順張り/逆張り（一部は片方向のみ） | ~830-1220 |
| + メタ次元（M1-M3） | Stage 1通過分に適用 | +100-200 |
| + Stage展開（逆シグナル、組合せ、動的閾値、遅延バージョン等） | Agent実行中に展開 | +150-300 |
| **合計** | | **推定1080-1770** |

この見積もりは仮定に依存する。特に「有意味な読み方の数」は事前に確定できない。
Agent実行後に実数を記録し、見積もりの精度を評価する。

## データパス

- 日足: `data/parquet/daily/{SYMBOL}.parquet`
  - カラム: Open, High, Low, Close, Volume, AdjClose
  - index: Date (datetime)
  - 銘柄: TQQQ, SQQQ, SOXL, SOXS, TECL, TECS, SPXL, TNA, TZA, VIXY
  - 期間: 2021-03-29 ~ 2026-03-27 (1256日)

- 5分足: `data/parquet/intraday/{SYMBOL}_5min.parquet`
  - カラム: Open, High, Low, Close（Volume なし）
  - index: date (datetime, tz=America/New_York)
  - 銘柄: TQQQ, SOXL, TECL, SPXL（結合済み197日）、SQQQ, SOXS, TNA, TZA（130日）
  - 期間: 2025-06-26 ~ 2026-03-27
  - 注意: 正規取引時間(9:30-16:00 ET)のみ。プレマーケットデータなし

- マクロ: `data/parquet/macro/{NAME}.parquet`
  - カラム: value（+ source, updated_at）
  - index: date (datetime)
  - 銘柄: VIX, VIX3M, VXN, HY_SPREAD, YIELD_CURVE, BRENT, USD_INDEX, US10Y, FEDFUNDS
  - 期間: 約1年 (2025-03 ~ 2026-03)

- DuckDB: `data/sensei.duckdb`
  - アクセス: `from src.db import SenseiDB`
  - テーブル: events(56), knowledge(19), predictions(2), regime_assessments(3)
