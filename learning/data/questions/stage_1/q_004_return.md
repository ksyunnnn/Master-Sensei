## Prompt

株価が $100 から $110 に上がった時の「リターン」は何 %？また、$110 から $100 に戻った時のリターンは -10% ではありません。正しくは何 %？計算を示してください。

## Rubric

- 上昇時: +10%
- 下落時: -9.09% (= 10/110)
- 下落率の基準は「現在の価格」
- 対称的ではない（上と下で同じ幅でも % は異なる）

## Explanation

**リターン** = **変化率 (%)** = **(終値 - 始値) / 始値 × 100**

- $100 → $110: リターン = (110 - 100) / 100 = **+10%**
- $110 → $100: リターン = (100 - 110) / 110 = -10/110 = **-9.09%**

重要な性質: **+10% と -10% は対称ではない**

$100 に 10% 戻すには +10% 必要だが、$110 から $100 に戻すには -9.09% で足りる。基準価格が変わっているため。

より dramatic な例:
- $100 → $50 (-50%) から元の $100 に戻すには **+100%** 必要

これがレバ ETF の Decay を理解する前提。

**日次リターン (daily return)**: 1 日の変化率
**累積リターン (cumulative return)**: 期間中の複利込み変化率

SOXL は「日次リターンの 3 倍」を日毎に再現する。累積では単純な 3 倍にならないのはこの非対称性 + 複利が原因。
