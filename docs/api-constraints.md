# API制約サマリー

各APIの詳細はADRを参照。ここは運用時のクイックリファレンス。

## Tiingo (価格OHLCV)

- Free tier: 50 req/時間, 1,000 req/日, 500 symbols/月
- 現在の消費: ~27 req/実行（日足10 + 5分足8 + マクロ9）
- IEX上限: 10,000 points/req（2026-03-26実測）
- レート制限: コード側で2秒間隔スロットリング + 429リトライ(60秒)
- 詳細: ADR-002, ADR-004

## FRED (マクロ指標 公式)

- 120 req/分。9シリーズで制限に対して十分余裕
- 公開遅延: 1-2日（シリーズによる）
- 詳細: ADR-002

## yfinance (マクロ指標 即時)

- 非公式API。レート制限は非公開（~2,000 req/時間の報告あり）
- 対応: VIX(^VIX), VIX3M(^VIX3M), Brent(BZ=F)
- FREDの遅延を補完。ProviderChainでフォールバック
- 詳細: ADR-005, ADR-006
